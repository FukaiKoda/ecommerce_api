What's under the hood

Key architectural concepts applied in this project

🏛️

Repository Pattern

Services never touch the database directly. A dedicated Repository layer mediates all Prisma calls, making the data layer swappable.

💉

Dependency Injection

Controllers receive a Service instance; Services receive a Repository instance through their constructors. Zero tight coupling.

🔐

Argon2id Hashing

Industry-standard password hashing using Argon2id — the winner of the Password Hashing Competition. No plain-text secrets.

🛡️

Joi Validation

All request body validation happens server-side in middleware. The frontend enforces nothing — trust is earned, not assumed.

🗄️

Prisma ORM

Type-safe database access with a declarative schema. Migrations are versioned and reproducible via prisma migrate.

⚡

Express 5 + Helmet

Built on Express 5 with rate limiting, CORS, and Helmet for sensible security defaults right out of the box.
